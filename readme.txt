Minecraft Server RCON Client by JeremySeq

Setup:
1. Open server.properties file in your server folder
2. Find 'enable-rcon' and make it true
3. Find 'rcon.password=' and make a password for your rcon server
4. Run rconClient.exe on client computer
5. Enter server address on prompt - find server address by typing ipconfig in cmd
6. Enter server rcon password on prompt
